# Living Map Japan (MVP)

日本の都市構造を「Before / After」で可視化する実験的プロジェクトです。

## Features
- 都市レイヤー（国家コア / 広域コア / 準拠点）
- Before / After シナリオ切替
- レイヤー別表示切替
- Leaflet + Next.js (App Router)

## Tech Stack
- Next.js 14
- React
- Leaflet
- TypeScript

## Development
```
npm install
npm run dev
```

## Status
MVP / Experimental
