# Living Map Japan

Living Map Japan は、日本の都市構造を  
**「Before / After」で可視化する試作的マッププロジェクト**です。

本プロジェクトは、将来予測や政策提言を目的としたものではありません。  
人口・労働力・財政といった制約条件の下で、  
**都市機能がどのように再配置され得るかを視覚的に理解するための判断材料**です。

---

## 主な機能

- 都市レイヤー（国家コア / 広域コア / 準拠点）の可視化
- Before / After シナリオ切替
- レイヤー別フィルタリング
- 地図ベースの直感的表示（Leaflet）

---

## 技術スタック

- Next.js（App Router）
- React
- TypeScript
- Leaflet
- Tailwind CSS
