# Living Map Japan – Documents

このディレクトリには、  
**Living Map Japan の設計思想・移行設計・合意形成設計**に関する文書が含まれています。

ここにある文書は「政策提言」や「結論」を示すものではありません。  
あくまで **可視化プロジェクトを設計するための前提・思想・構造** を整理したものです。

---

## ドキュメント構成

### 1. Vision（G-1）
**何を守り、何を捨てるのか**

- 人口減少と労働制約を前提にする
- すべてを救わないことを前提にする
- 地域ではなく「機能」を整理する

→ 都市レイヤー設計の思想的土台

📄 `vision.md`

---

### 2. Transition（G-2）
**どうやって変えるのか**

- 強制しない
- 突然変えない
- 失業を生まない
- 仕事だけを消す

→ Before / After シナリオ設計の根拠

📄 `transition.md`

---

### 3. Consensus（G-3）
**どう説明し、どう合意を壊さないか**

- 説得しない
- 正しさを主張しない
- 責任を一人に集めない
- 地図に語らせる

→ UI 表現・文言設計・公開範囲判断の指針

📄 `consensus.md`
